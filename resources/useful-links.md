## 🔧 Tools & Utilities

### VS Code Extensions
- **HashiCorp Terraform** - Official extension with syntax highlighting
- **Terraform Autocomplete** - Auto-completion support
- **Terraform Docs** - Documentation viewer

### CLI Tools
```bash
# tfenv - Terraform version manager
git clone https://github.com/tfutils/tfenv.git ~/.tfenv

# terraform-docs - Generate docs from modules
git clone https://github.com/terraform-docs/terraform-docs

# tflint - Terraform linter
git clone https://github.com/terraform-linters/tflint

# checkov - Security scanner
pip install checkov
```

## 🎯 Exam Tips & Strategies

### Study Strategy
1. Read all 9 objectives thoroughly
2. Complete hands-on labs
3. Take practice exams
4. Review wrong answers
5. Focus on weak areas

### Exam Day Tips
- Get good sleep before exam
- Test your setup beforehand
- Have ID ready
- Read questions carefully
- Flag uncertain questions for review
